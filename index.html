The Engine

Hybrid AI + rule engine.
Statutory violations never left to hallucination.

Two layers. Each doing what it does best.

A Python rule engine runs before the AI call — catching Criminal Code interest caps, Ontario non-compete bans, Construction Act prompt payment violations, and provincial statutory requirements deterministically. These are not AI judgment calls. They are binary rule checks that are always correct.

The AI then handles what requires contextual judgment — clause interpretation, enforceability assessment, benchmark comparison, and replacement drafting — with the statutory ground truth injected as context the AI cannot contradict.

The result: Statutory compliance is guaranteed. Contextual analysis is enhanced. Neither is left to chance.

FAQ

Common questions.

Is this legal advice?

No. ContractShield provides an AI-powered compliance audit and strategic memo. It is a powerful information tool, not a substitute for a licensed Canadian lawyer. For the professional desktop application, every output is structured for review by qualified counsel — professional responsibility remains with the lawyer.

Is my contract data safe?

Yes. We use PIPEDA-compliant processing — personal information is detected and redacted before analysis. Data is encrypted in transit and verbatim contract text is never stored permanently. For the desktop application, all processing happens locally on your machine — no data leaves your office.

Why $5? What's the catch?

No catch. We've automated the routine first-pass review process. By focusing on high-speed statutory checks and AI-driven analysis we pass the efficiency savings directly to you. The $5 price IS the disruption. Law firms pay more for the professional-grade desktop application with unlimited contracts and the full feature set.

What types of contracts do you support?

Freelance agreements, employment contracts, vendor agreements, NDAs, service agreements, construction contracts, commercial leases, and most standard Canadian business and legal contracts. The desktop version adds support for complex M&A documents, 100+ page agreements, and multi-document transaction suites.

What provinces and jurisdictions are covered?

The web version covers federal Canadian law including Criminal Code s.347, PIPEDA, and Ontario-specific statutes. The professional desktop version covers 10+ provincial jurisdictions — Ontario ESA/Bill 27, Quebec CCQ/Bill 96, BC PIPA, Alberta, and more — with province-specific statutory rules applied per contract.

How does the hybrid engine work?

A Python rule engine runs before the AI call — catching Criminal Code interest caps, Ontario non-compete bans, Construction Act violations, and other statutory issues deterministically. These are binary rule checks, not AI judgment calls. The AI then handles contextual analysis with the statutory ground truth injected as context it cannot contradict.

Can law firms use the $5 web version?

Yes — many lawyers and paralegals use the $5 web version for quick NDA and standard agreement checks. The professional desktop application is for firms that need all three output modes, all depth levels, matter history, Law Society audit export, multi-user access, and complete local processing with client data that never leaves the office.
